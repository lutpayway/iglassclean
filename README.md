# iglassclean
This project develops an IoT-based automated high-rise glass cleaning module. It integrates weather sensors and a timer system for safer operation, with monitoring and control available through a web dashboard and a Flutter mobile app for remote, real-time access
Overview

This project develops an IoT-based automated high-rise glass cleaning module. The system integrates weather sensors and a timer mechanism to ensure safe and efficient operation. Monitoring and control are accessible via web dashboard and a mobile app built with Flutter.

Features

Automated glass cleaning system

Weather sensor integration for safety

Timer-based operation control

Web-based monitoring dashboard

Mobile app control (Flutter)

Project Structure

IoT hardware & sensor integration

Web monitoring system

Mobile application interface

Supporting datasets/assets
